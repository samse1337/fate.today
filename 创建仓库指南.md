# 创建 fate.today 私有仓库指南

## 方式一：GitHub 网页创建（推荐，无需安装工具）

### 1. 在 GitHub 创建私有仓库

1. 打开 https://github.com/new
2. **Repository name** 填：`fate.today`
3. 选择 **Private**（私有）
4. **不要**勾选 "Add a README file"
5. 点击 **Create repository**

### 2. 在本地项目目录执行

在项目根目录（有 package.json 的 app 文件夹）打开终端，执行：

```powershell
# 初始化 git（如果还没有）
git init

# 添加所有文件
git add .

# 首次提交
git commit -m "Initial commit"

# 添加远程仓库（把 YOUR_USERNAME 换成你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/fate.today.git

# 推送到 main 分支
git branch -M main
git push -u origin main
```

---

## 方式二：使用 GitHub CLI（需先安装）

如果已安装 [GitHub CLI](https://cli.github.com/) 并登录：

```powershell
cd "你的项目路径\app"

git init
git add .
git commit -m "Initial commit"

gh repo create fate.today --private --source=. --push
```

一条命令即可完成创建私有仓库并推送。

---

## 推送后启用 GitHub Pages

1. 进入仓库 → **Settings** → **Pages**
2. **Source** 选择 **GitHub Actions**
3. 推送代码后会自动部署，访问地址：
   - `https://YOUR_USERNAME.github.io/fate.today/`
